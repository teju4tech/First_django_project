# First [django] web project



##  Reference of project

#### Get all items

```http
  GET /framwork/items
```

| Parameter | Type     | Description                |
| :---- | :------- | :------------------------- |
| `Ref_key` | `string` | **Required**. Your Reference key |

#### Get item

```http
  GET /framwork/items/${id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `id`      | `string` | **Required**. Id of item to fetch |
